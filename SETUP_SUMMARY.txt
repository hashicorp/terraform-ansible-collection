If something isn't working:

1. RUN VALIDATION SCRIPT
   $ python3 scripts/validate_setup.py
   This will identify most configuration issues

2. CHECK DOCUMENTATION
   • DEVELOPMENT.md → Troubleshooting section
   • scripts/README.md → Troubleshooting section
   • GETTING_STARTED.md → Troubleshooting section

3. COMMON ISSUES

   "module not found" in tests:
   $ source .venv/bin/activate
   $ pip install -r requirements.txt -r test-requirements.txt

   Collection not discoverable:
   $ ansible-galaxy collection install . --force

   Virtual environment issues:
   $ rm -rf .venv
   $ ./scripts/setup.sh

   Permission denied on script:
   $ chmod +x scripts/setup.sh
   $ chmod +x scripts/validate_setup.py

4. STILL STUCK?
   • Check validation output for specific issues
   • Review relevant documentation
   • Open issue with validation output and error message
